Introduction to Python

The Python Story

Python was created by Guido van Rossum and first released in 1991. Named after Monty Python's Flying Circus (the comedy show), Python was designed to be:

  • Easy to learn with clear, readable syntax
  • Versatile for various programming tasks
  • Powerful with extensive standard libraries

Python is an experiment in how much freedom programmers need. Too much freedom and nobody can read another's code; too little and expressiveness is endangered. — Guido van Rossum

Python in Action

Here's your first Python program - a simple "Hello World":

# Simple Python Program
                    print("Hello, World!")  # This line prints text

And a more practical example - a temperature converter:

# Fahrenheit to Celsius Converter
                    fahrenheit = float(input("Enter temperature in Fahrenheit: "))
                    celsius = (fahrenheit - 32) * 5/9
                    print(f"{fahrenheit}°F = {celsius:.2f}°C")

Key Python Features

Language Characteristics

  • Interpreted: No compilation needed
  • Dynamic typing: No variable type declarations
  • Indentation-based: Uses whitespace for code blocks
  • Multi-paradigm: OOP, functional & procedural

Ecosystem Strengths

  • Rich standard library: "Batteries included" philosophy
  • Huge ecosystem: 300,000+ packages (PyPI)
  • Cross-platform: Runs on Windows, Mac, Linux
  • Community: 11M+ developers worldwide
