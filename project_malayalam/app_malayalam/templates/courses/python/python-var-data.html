Variables and Data Types in Python

Understanding Variables

Variables are containers for storing data values. Unlike other languages, Python doesn't require explicit declaration of variables.

# Variable assignment examples
                    name = "Alice"  # String
                    age = 25        # Integer
                    height = 5.9    # Float
                    is_student = True  # Boolean

Variable Naming Rules

  • Must start with a letter or underscore (_)
  • Can only contain letters, numbers, and underscores
  • Case-sensitive (age and Age are different)
  • Cannot be a Python keyword (if, for, etc.)

Python Data Types

Python has various built-in data types:

Basic Types

  • int: Integer numbers (3, -10, 0)
  • float: Decimal numbers (3.14, -0.001)
  • str: Text ('hello', "world")
  • bool: Boolean (True, False)

Collection Types

  • list: Ordered, mutable [1, 2, 3]
  • tuple: Ordered, immutable (1, 2, 3)
  • dict: Key-value pairs {'name': 'Alice'}
  • set: Unordered, unique {1, 2, 3}

Type Conversion

Python allows converting between different types:

# Type conversion examples
                    num_str = "123"
                    num_int = int(num_str)  # Convert to integer
                    num_float = float(num_str)  # Convert to float

                    price = 19.99
                    price_str = str(price)  # Convert to string

                    is_valid = 1
                    bool_valid = bool(is_valid)  # Convert to boolean (True)

Dynamic Typing Tip

Python uses dynamic typing - variables can change type. Use type() to check a variable's type: print(type(age))

Practical Examples

User Information

name = input("Enter your name: ")
age = int(input("Enter your age: "))
print(f"Hello {name}, \
you are {age} years old!")

Temperature Converter

celsius = 37.5
fahrenheit = (celsius * 9/5) + 32
print(f"{celsius}°C = {fahrenheit}°F")
